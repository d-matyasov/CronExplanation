Проект реализован с использованием Spring Boot Framework и Thymeleaf.
Функционал: перевод cron-выражения в читаемый вид на русском языке.
Для открытия в адресной строке браузера ввести адрес сервера с портом (пример: localhost:8080).

Примеры выражений:
0-30/5,45 */5 * L SEP-dec monl
@monthly
0 */5 * 3W 3,6 2#2
